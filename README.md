## Hillshade map of Ñuble
Topographic map of Ñuble Region, Chile, developed with the software QGIS and using digital elevation model (DEM) data from ALOS satellite's PALSAR sensor (IDE, 2016). The coordinate reference system (CRS) was set to EPSG 32719 (WGS 84 / UTM zona 19S).
 
A high resolution version of the map is available [here](/figure/Nuble_cartography.png)

### References
Infraestructura de Datos Geoespaciales (IDE Chile). 2016. DEM Alos Palsar Región del Ñuble. Available at: [geoportal.cl](https://geoportal.cl/geoportal/catalog/35436/DEM%20Alos%20Palsar%20Regi%C3%B3n%20del%20%C3%91uble)
