# Hillshade map of Ñuble, Chile
Topographic map of Ñuble Region, developed with the software QGIS and using digital elevation model (DEM) data from ALOS satellite's PALSAR sensor (IDE, 2016).

### Referencias
Infraestructura de Datos Geoespaciales (IDE Chile). 2016. DEM Alos Palsar Región del Ñuble. Disponible en: [geoportal.cl](https://geoportal.cl/geoportal/catalog/35436/DEM%20Alos%20Palsar%20Regi%C3%B3n%20del%20%C3%91uble)
